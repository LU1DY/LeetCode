# 🧠 Desafios do LeetCode

Este repositório é destinado a resoluções de desafios do **LeetCode**.  
Meu principal objetivo com ele é **treinar minha lógica de programação** e reforçar a base das três linguagens que estudo: **Python**, **Java** e **JavaScript**.

Cada exercício é nomeado seguindo o padrão:


Exemplo: `ex_1`, `ex_13`, `ex_217` etc.

---

## 🎯 Objetivos do Repositório

- Melhorar minha capacidade de resolução de problemas;  
- Fixar conceitos importantes das três linguagens;  
- Criar um histórico organizado da minha evolução.

---
